Booster 4.0.0 版本发布

2021-08-22 · Johnson Lee · Computer Science / Open Source / Booster

Booster 又双叒叕发布了新的版本—— v4.0.0，本次更新内容如下：

  1. 支持 AGP 7.0
  2. 开放 AGPInterface
  3. 优化 AGP 版本适配，避免加载不兼容的 AGPInterface 实现
  4. 废弃获取 AGP 内置 Task 的扩展属性，由新增的 TaskProvider 相关的 API 替代
  5. 由于 Gradle Task API 的兼容性问题，Booster 不再支持 AGP 3.3 以下的版本
  6. 资源去重和压缩相关的 Task 的缓存行为发生变化

参见：API Rereference
