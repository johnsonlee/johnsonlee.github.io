Booster 4.8.0 版本发布

2022-05-05 · Johnson Lee · Computer Science / Open Source / Booster

Booster 又双叒叕发布了新的版本—— v4.8.0，本次更新内容如下：

  2. 新增 API BaseVariant.mergeNativeLibsTaskProvider by punkisnotdead3

  4. 优化 booster-task-check-snapshot, booster-task-list-artifact, booster-task-list-permission, booster-task-list-shared-library
  5. 调整 booster-task-graph 生成的图的方向
