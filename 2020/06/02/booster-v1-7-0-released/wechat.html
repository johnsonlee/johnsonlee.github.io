Booster v1.7.0 发布

2020-06-02 · Johnson Lee · Computer Science / Open Source / Booster

Booster 又双叒叕发布了新的版本—— v1.7.0，本次更新内容如下：

  1. 修复 booster-cha 模块传递依赖问题
  2. 重构 booster-transform-util 模块，新增实用 API
  3. 重构 booster-android-gradle-api 模块，优化 Android Gradle Plugin 版本适配
  4. 优化 Service Provider Interface 加载
  5. 增加 dokka 文档注释，部分 Java 模块用 Kotlin 重写

原文链接：https://johnsonlee.io/2020/06/02/booster-v1-7-0-released/
