Booster 3.0 正式发布，支持 AGP 4.1

2020-11-09 · Johnson Lee · Computer Science / Open Source / Booster

在 Booster 3.0.0 版本中，所有模块和特性已完全支持 Android Gradle Plugin 4.1.0，本次更新内容如下：

  1. 支持 Gradle plugins DSL

    plugins {
    id 'com.didiglobal.booster' version '3.0.0'
}

  2. 重新设计 Android Gradle Plugin 的兼容性适配方案

  3. 适配 Android Gradle Plugin 4.1.0

  4. 修复以往版本 Android Gradle Plugin 兼容性问题

  5. 增加 Android Gradle Plugin 集成测试，保证 API 的兼容性和稳定性

原文链接：https://johnsonlee.io/2020/11/09/booster-v3-0-released/
