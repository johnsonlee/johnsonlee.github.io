兴趣不是最好的老师

Oct 7, 2020 | Independent Thinking | 2.1k 字 | 7 分钟

每当我跟 IT 圈内的朋友说我是生物专业的时候，对方都会大吃一惊，身边的朋友都说我是一个被代码耽误的厨师 😂 ，说心里话，我确实对烹饪挺感兴趣的，很多不喜欢做饭的朋友表示无法理解，一方面可能是基因遗传的缘故，另一方面可能是大量的实践（从我6岁开始，我妈就开始教我煮饭，在我13岁的时候便能独立待客了）。尽管煎炒煮炸难不倒我，但 20 多年的烹饪经历，我始终没能成为真正的大厨，充其量只能算朋友圈的大厨罢了，烹饪之于我，无论是中式还是西式，基本功都很熟练，为什么却一直止步于此呢？

成就感

如果说兴趣爱好，烹饪只是我的其中一个爱好，对它之所以感兴趣，可能主要还是因为能从中获得成就感和满足感。在我学生时代，每到暑假，家里做饭的活儿基本上都是我全包了，有一次，准备炒鲊辣椒（湖南人最爱吃的菜之一），炒鲊辣椒基本上都是清炒，不放别的辅料，那天我经过田间的小路，看到路边的青黄豆，突发奇想：“要是红红的鲊辣椒中配上这青青的黄豆，味道肯定好极了”，一方面是视觉上的冲击力，另一方面是不一样的口感，鲊辣椒炒完比较干，带有颗粒感，如果能配上青黄豆的松软的嚼觉感，那效果肯定很赞，于是，剥了小半盘青黄豆，搭着鲊辣椒一起炒，晚上，我妈尝到我的创意之作，一边极力夸赞，一边问我跟谁学的这道菜，我说：“这是我自创的”！

除了烹饪，吉他也算是我的一个兴趣爱好，大学时代为了打发无聊的时光，在宿舍开始对着视频教程自学吉他，对着卫生间的镜子练习弹唱，终于能够跟着节拍完全整的弹上一曲。虽然自始至终我对 5 线谱是一窍不通，但只要有 6 弦谱，我就可以跟着弹出来，从古典到民谣，再到插电，都研究了一番，基本都是先看视频，再找 GTP 曲谱，再导入 Guitar Pro 跟着节拍反复练习，为此还专门花了几百大洋买了 Guitar Pro 的 License ，2013 年的某个夜晚，录下了最为熟练的几曲弹唱，每每回放，都沉浸在熟悉的旋律中无法自拔。

挫败感

很多朋友得知我是生物专业后，都很好奇，究竟是什么原因让我选择了 IT 而不是生物？其实，当初我选择生物专业，也是因为我对生物很感兴趣，其实我对医学更感兴趣，只是家人不太支持学医，所以，退而求其次，选择了生物，毕竟生物专业毕业的也算半个医生了。因为感兴趣，我的有机化学、生物化学都学得不错。

有一次，在生物化学课上，讲到了 DNA/RNA 复制的那一章节，老师说道：“DNA 链的复制总是从 5’ 端（脱氧核糖核酸的 5 号碳原子位）向 3’ 端（脱氧核糖核酸的 3 号碳原子位）进行”，我当时就很疑惑，为什么是 5’ 端到 3’ 端，而不是 3’ 端到 5’ 端？带着这个疑问，我在课后找到了这位老师，估计这位老师从来没有遇到像我这样提出这种问题的学生，一时很尴尬，然后告诉我，这个现象是实验得出来的，至于为什么，她也不知道。

又是一节生物化学课，讲到了基因的表达（蛋白质的合成），又说到了一个结论：多肽链的合成是从 N 端向 C 端进行，我又向这位老师提出了和之前类似的问题 —— 为什么多肽链的合成是从 N 端和 C 端进行的？最后我得到的结果还是跟之前一样 —— 实验得出来的。整个大学期间，这两个问题一直困扰着我。

在一次生化实验课上，练习 DNA 的 PCR（聚合酶链式反应）技术，说人话就是 DNA 扩增，人工复制 DNA（比如常见的通过微量样本做 DNA／RNA 分析时，采样的量极期微小，根本不够一次实验，如果要进行分析，会先对样本 DNA/RNA 进行扩增到足够多的量），而 PCR 技术需要用到一种关键的引物（类似中药中的药引子），只有它才能激活整个 PCR 的过程，这个引物本身也是一段核苷酸片段，大概只有几十个碱基，因为短，所以结构也很简单，但是这个试剂当时国内造不出来（或者纯度达不到要求，导致错误率很高），自此之后，我便对生物失去了研究的热情，现在回想起来，如果当初我的问题能够被合理的解答，我也就不会对生物失去信心。

最好的老师

后来，我开始学习 Web 编程，刚开始也是看视频自学，也有向网友请教，但一直没有领悟到精髓，只会抄别人写过的代码，不知道如何自己创造代码，直到有一天，一位姓胡的老师给我了一本 《JavaScript 权威指南第 5 版》的 CHM 格式的文件，我这才知道，原来还有一种叫做 API References 的东西，从此，我便打开了新世界的大门。

常言道：兴趣是最好的老师，但我却发现，事实并非如此，同样都是兴趣爱好，有些学有小成，有些则是无疾而终，而有些却是一生的事业，为什么差别是如此之大，究竟是什么决定了我们成长的高度？

拿烹饪来说，一位合格的烹饪大师难道只是精通烹饪吗？我个人认为，精通烹饪技巧只能让其成为一个主厨，而从主厨到大师，要习得的技能更多的是烹饪之外的知识体系，比如：营养学、植物学、动物学、微生物学等等，只有掌握了食材的结构和质地以及制做工艺，才能去创造新的烹饪技术，而不是拿着菜谱照猫画虎。

而当年一直困扰着我的 2 个生物化学问题的答案也是出乎意料的简单 —— 能量成本。

因为合成 DNA 的原材料是 dNTP (脱氧三磷酸核苷，dATP, dTTP, dGTP, dCTP 的总称)，5’ 端是磷酸基团，释放掉 β 和 γ 磷酸基团，既可以为 DNA 复制提供能量，剩下的磷酸酯还能连接 3’ 端的羟基，如果反方向的话，要先破坏掉 5’ 端的磷酸基团，与 3’ 端的羟基连接，破坏磷酸基团需要消耗更多的能量，而磷酸基团被破坏后，不一定还能保存完整的 β 和 γ 磷酸基团来释放产生能量，所以，反方向消耗的能量更高，用经济学的思维来解释就是生产成本更高。

而 dNTP 则是由葡萄糖和脂肪酸合成而来，这也就将糖代谢与脂肪代谢过程联系了起来，其实，整个链路都是为 DNA/RNA 复制（新陈代谢）服务，这才是整个生物化学的知识框架。

当年，老师只是照本宣科的把各个知识点讲了一遍，并没有串起来形成一个知识体系，对于生命的奥秘，只能从生命科学导论上的介绍了解一些常识性的知识，而真正的精华是在于将整个体系的融汇贯通，就像功夫熊猫中的阿宝，在没学会气功之前，只能算是很能打的大侠，当他真正掌握了气功（知识框架）后，最终成为了一代宗师。
